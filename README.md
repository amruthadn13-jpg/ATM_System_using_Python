# ATM_System_using_Python

# 🏦 Day 6 – ATM System (Python Functions)

This project is a Python-based **ATM System** that simulates basic banking operations using functions. It allows users to check balance, deposit money, withdraw money, and exit through a menu-driven interface.

---

---

## 📌 Features

* Check account balance
* Deposit money
* Withdraw money with validation
* Menu-driven system
* Continuous operation using loop
* Clean and modular code using functions

---

## 🧠 Concepts Used

* Functions
* Loops (`while`)
* Conditional statements (`if`, `elif`, `else`)
* User input handling
* Return values
* Basic arithmetic operations

---

## ⚙️ How It Works

1. Program starts with an initial balance

2. Displays ATM menu options:

   * Check Balance
   * Deposit Cash
   * Withdraw Cash
   * Exit

3. Based on user choice:

   * Calls respective function
   * Updates balance if needed

4. Runs continuously until user selects Exit

---

## 💡 Functions Used

### 1. Check Balance

Displays current account balance.

### 2. Deposit Cash

* Takes amount as input
* Adds to balance
* Returns updated balance

### 3. Withdraw Cash

* Takes amount as input
* Checks if balance is sufficient
* Deducts amount if valid
* Returns updated balance
